# Сравнительный анализ ответов: Gemini vs YandexGPT

| Параметр оценки | Gemini (Google) | YandexGPT (Yandex) |
| :--- | :--- | :--- |
| **Техническая логика** | **Отличная.** Подробно разобрал разницу между методом бурильщика и методом ожидания. | **Хорошая.** Описал общие шаги герметизации устья без глубоких расчетов. |
| **Нормативная база** | Общие мировые правила безопасности. | **Высокая.** Учтены требования экологического надзора РФ и ГОСТы на сорбенты. |
| **Формат ответа** | Детальные технические списки. | Четкие инструкции для заполнения журналов и актов. |

**Анализ Кейса №1:** Gemini верно расставил приоритеты при ГНВП, сделав упор на стабилизацию давления в скважине.

**Анализ Кейса №2:** YandexGPT предоставил готовый перечень документов, необходимых для фиксации экологического инцидента на месторождении в РФ, что критически важно для ИТР.
