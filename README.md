# Сравнительный анализ ИИ-инструментов в задачах Промбезопасности и ГНВП
**Автор:** Курскиев М.В., группа БНГа-o24 (ЮРГПУ (НПИ))

---

## 1. Обзор проекта
Данный репозиторий содержит результаты практического исследования возможностей современных языковых моделей (LLM) при решении критических задач в нефтегазовой отрасли. Основное внимание уделено сценариям управления скважиной и соблюдению экологических стандартов РФ.

## 2. Структура репозитория
* `cases/` — подробное описание выбранных инженерных сценариев.
* `prompts/` — точные формулировки запросов, использованные в ходе тестирования.
* `results/` — визуальные подтверждения ответов нейросетей (скриншоты).
    * `foreign_bot/` — ответы модели Gemini (Google).
    * `domestic_bot/` — ответы модели YandexGPT (Yandex).
* `analysis/` — аналитическая записка со сравнительными таблицами.

## 3. Описание кейсов исследования

### Кейс №1: ГНВП (Газонефтеводопроявление)
* **Ситуация:** Зафиксирован перелив раствора при подъеме инструмента.
* **Задача для ИИ:** Составить алгоритм герметизации устья и обосновать выбор метода глушения скважины (метод бурильщика против метода ожидания и утяжеления).

### Кейс №2: Экологическая безопасность на буровой
* **Ситуация:** Аварийный разлив бурового раствора на водной основе.
* **Задача для ИИ:** Разработать план ликвидации последствий, подобрать сорбенты и указать перечень необходимых документов согласно нормам РФ.

## 4. Методология исследования
1.  **Формирование промптов:** Созданы узкоспециализированные запросы, требующие от ИИ понимания терминологии БНГА.
2.  **Тестирование:** Запросы отправлены обоим чат-ботам одновременно в новых сессиях.
3.  **Сравнение:** Ответы оценивались по глубине технической проработки и релевантности для российского законодательства.

## 5. Итоговое заключение
В ходе исследования выявлено:
* **Gemini** эффективен для оперативной технической поддержки, так как глубже анализирует физические параметры (давление, объемы притока).
* **YandexGPT** является приоритетным инструментом для работы с административными и экологическими регламентами, принятыми на территории РФ.
